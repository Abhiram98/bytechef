---
title: Environment Variables
description: Configuration options for ByteChef through environment variables
---

ByteChef can be configured using environment variables. This page documents all available environment variables, organized by category.

## AI Configuration

| Environment Variable | Description | Default Value |
|----------------------|-------------|---------------|
| `BYTECHEF_AI_COPILOT_ENABLED` | Enable or disable the AI copilot feature | `false` |
| `BYTECHEF_AI_COPILOT_PROVIDER` | The AI provider to use for copilot (OPENAI, ANTHROPIC) | `OPENAI` |
| `BYTECHEF_AI_COPILOT_OPENAI_APIKEY` | OpenAI API key for copilot (sensitive) | - |
| `BYTECHEF_AI_COPILOT_OPENAI_CHAT_OPTIONS_MODEL` | OpenAI model to use for chat | - |
| `BYTECHEF_AI_COPILOT_OPENAI_CHAT_OPTIONS_TEMPERATURE` | Temperature setting for OpenAI chat | - |
